HTML Commands

In the following we have a list of HTMl and Semantic Markup commands with the links to their example as we move forward. In each example we see the command added and how it changes the webpage.

  • < doctype > It is not a html tag but shows the version of the html to the browser.
  • < html > The first thing in html file to show that we are working in html.
  • < head >Provides general information about the webpage.
    • < title >Creates the title of the page shown on top of the webpage.
    • < meta > General information about the type of symbols to be used for webpage.
  • < h1...h6 > Headings from largest to smallest that show the importance of things.
  • < body >Represent the contents of the html file.
  • < p >Represent a paragraph of the text.
  • < b >Bolden the text without putting any importance to it.
  • < i >Set off the text by italizing it.
  • < sup >Displays text as the superscript often higher and smaller.
  • < sub >Displays text as the subscript often lower and smaller.
  • < br/ >Breaks the line.
  • < hr/ >Creates thematic break between paragraph level elements.

Semantic Markups

  1. < Strong >It shows that its content has strong importance.
  2. < em >Indicates emphasis that subtly changes the meaning of a semtence.
  3. < blockquote > Is used for longer quotes that take up an entire paragraph.
  4. < q > Is used for shorter quotes that sits within one paragraph.
  5. < abbr titie="" > To use for abbreviation.
  6. < acronym title="" > To use for acronyms.
